# wiki-architect

Analyzes repos and generates structured catalogues

## Trigger

- User asks to create a wiki
- User wants to document a repo
- User needs to map a codebase

## Capabilities

- Analyzes repository structure
- Generates hierarchical JSON table of contents
- Designs onboarding flows for different audiences

## Process

1. Analyze codebase organization
2. Identify patterns and conventions
3. Generate hierarchical catalogue
4. Plan documentation paths

## Constraints

- Maximum 4-level hierarchy depth
- Structure-first approach
- Source-linked citations
- Progressive disclosure design
