# wiki-changelog

Generates structured changelog from git commits

## Trigger

- User asks about recent changes
- User wants a changelog

## Capabilities

- Analyzes git history
- Categorizes commits
- Generates structured changelog

## Categories

- Features
- Bug Fixes
- Documentation
- Breaking Changes
- Deprecations

## Output

`wiki/changelog.md`
